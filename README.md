# ☕ Coffee Peace

Uma landing page moderna com foco em design minimalista e experiência do usuário, inspirada na tranquilidade e energia que o café proporciona.

## 🛠️ Tecnologias

- HTML5
- CSS3

## 🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar desenvolvimento front-end, com foco em:

- Estruturação semântica com HTML
- Estilização moderna com CSS
- Design responsivo para desktop com CSS

- Layout moderno e minimalista
- Responsividade para diferentes dispositivos
- Interface intuitiva
- Seções organizadas
